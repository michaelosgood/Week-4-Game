Gold Coins Collector

Rules:

  • A random number will be generated at the beginning of the game.
  • There are four gold coins that will have an undisplayed value associated with it.
  • Clicking on the gold coin will add a number to your total score.
  • The value associated with each gold coin will add up and the total sum will display below.
  • The goal is to to add the coins up to a value equivalent to the randomly generated number.
  • You lose if your total exceeds the value of the randomly generated number.
  • The value of each gold coin will reset with each new game
